# F1 Console - Copilot Instructions

## Project Overview
F1 Console is a Formula 1 data analysis and visualization project using FastF1.

This project is a full-stack JavaScript F1 Console application designed for F1 race data analysis and visualization.

## Architecture

The project follows a monorepo architecture with:

- `packages/frontend/`: React-based web application
- `packages/backend/`: FastAPI API server
- `fastf1-examples/` - Example scripts for analyzing and plotting F1 data
- `docs/` - Project documentation including functional requirements

Testing coverage spans unit, integration, and UI end-to-end validation across frontend and backend behaviors.

## Technology Stack

### Frontend

- React 18
- Material-UI (MUI) v5 - Modern component library
- @mui/icons-material - Icon components
- @emotion/react & @emotion/styled - CSS-in-JS styling (required by MUI)
- React Query (TanStack Query) v5 - Data fetching and state management
- React Testing Library - Component testing
- Playwright - UI end-to-end testing for critical user journeys
- ESLint for code quality

### Backend

- FastAPI - Python web framework for API development
- Pytest for testing
- ESLint for code quality


## Key Technologies
- Python
- FastF1 library for F1 data
- Plotting and visualization libraries
